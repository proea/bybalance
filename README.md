БайБаланс
=========

Определение баланса счетов сотовых операторов и интернет провайдеров Республики Беларусь.

В этой версии поддерживаются:

* МТС
* Velcom
* Life :)
* Деловая сеть (iDom)
* TCM
* НИКС
* Шпаркі Дамавік
* Соло
* Телесеть
* NetBerry
* Космос ТВ

Работает на устройствах iOS 5 и новее

* iphone >= 3gs
* ipad
* ipod touch >= 3
